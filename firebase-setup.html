🔥 Firebase Sozlash

Quyidagi 3 ta qadamni bajaring — 2 daqiqa vaqt oladi

1

Firebase loyiha yarating

  1. Bu havolani bosing: console.firebase.google.com
  2. Google hisobingiz bilan kiring
  3. "Add project" (yoki "Loyiha qo'shish") tugmasini bosing
  4. Loyiha nomi: labo-taxi deb yozing → "Continue"
  5. Google Analytics: o'chirib qo'ying → "Create project"
  6. Loyiha yaratilgandan keyin "Continue" bosing
2

Firestore Database yarating

  1. Chap menyudan "Build" → "Firestore Database" tanlang
  2. "Create database" tugmasini bosing
  3. "Start in test mode" ni tanlang → "Next"
  4. Region: eng yaqin regionni tanlang (masalan europe-west1) → "Enable"
3

Web App qo'shing va Config ni oling

  1. Loyiha sahifasida tepadagi ⚙️ (gear) → "Project settings"
  2. Pastga aylantiring → "Your apps" → </> (Web) ikonini bosing
  3. App nomi: labo-taxi → "Register app"
  4. Ekranda paydo bo'ladigan firebaseConfig ni ko'chirib, pastdagi maydonga joylashtiring:
⚠️ Quyidagi ko'rinishdagi kodni ko'chirib oling:

const firebaseConfig = { apiKey: "...", authDomain: "...", ... };

✅ Firebase sozlandi!

Endi admin panelni oching va haydovchi qo'shing.
APK ni qayta build qilish kerak.
